# Tarot Shuffle - タロット展開アプリ

スマートフォンとPCに対応した、インタラクティブなタロット展開アプリケーション。

## 🎴 主な機能

### スプレッド（展開方法）
- **ワンカード** - 1枚のカードでシンプルな占い
- **3カード** - 過去・現在・未来の3カード展開
- **クロス** - 5枚のケルトの十字展開
- **自由配置** - カスタム枚数での自由な配置

### カード設定
- **逆位置** - カードの逆向きを有効化
- **小アルカナ** - 大アルカナと小アルカナの選択
- **背景テーマ** - オフホワイト・ネイビー・ブラウン

### ビジュアル機能
- 展開されたカードを選択してキーワードを表示
- カード情報の詳細パネル
- Instagram用4:5ビジュアルシェアリング機能

## 📱 レスポンシブデザイン

### スマートフォン対応
- 390px幅でも3枚のカードが横並び
- コンパクトなツールバーレイアウト
- タッチフレンドリーなボタンサイズ

### PC対応
- 大画面での快適な操作性
- クラシカルなレイアウト

## 🛠 技術スタック

- **HTML5** - セマンティックマークアップ
- **CSS3** - グリッドレイアウト、メディアクエリ
- **Vanilla JavaScript** - フレームワーク不使用

## 🎨 デザイン特性

- **色彩スキーム** - タロットの伝統を反映したエレガントなカラーパレット
  - ゴールド、テール、クラレット、バイオレット
- **タイポグラフィ** - Georgiaとモダンなサンセリフフォントの組み合わせ
- **グラデーション背景** - 落ち着いた雰囲気を演出

## 🚀 使い方

1. `index.html`をブラウザで開く
2. スプレッド方法を選択
3. 「並べる」ボタンをクリック
4. 展開されたカードをクリックしてキーワードを確認
5. Instagramボタンでシェア用画像を保存

## 🔧 デバッグモード

URL末尾に `?debug=1` を追加するとデバッグモードが有効になります：
- 特定のカードを固定表示
- デバッグ用のカード選択ドロップダウン
